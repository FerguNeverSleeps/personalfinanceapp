How to use Rules (help)

Rules auto-categorize transactions by matching Merchant or Description.

Match type:

  • Contains (recommended): keyword appears anywhere.
  • Regex (advanced): only if you need patterns.

Priority: lower runs first. When multiple rules match, the first match wins.

  • 10 = very specific (spotify/uber/netflix)
  • 100 = normal
  • 200 = broad catch-all

Example: Merchant contains uber → Transport (priority 10)
